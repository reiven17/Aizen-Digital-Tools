# Assets – Aizen Digital Tools

Esta carpeta contiene todos los recursos gráficos del sitio.  
Sube los archivos en la carpeta correspondiente respetando los nombres exactos indicados.

---

## 📁 `iconos/`

Iconos de redes sociales y navegador usados en el sitio.

| Archivo            | Uso                          |
|--------------------|------------------------------|
| `tiktok.png`       | Botón de red social (footer) |
| `facebook.png`     | Botón de red social (footer) |
| `whatsapp.png`     | Botón de red social (footer) |
| `favicon-32.png`   | Favicon del navegador        |
| `apple-touch-icon.png` | Acceso directo iOS      |

**Tamaño recomendado:** 64 × 64 px o 128 × 128 px, fondo transparente (PNG).

| `paypal.png`          | Método de pago (chip)                             |
| `visa.png`            | Método de pago (chip)                             |

- `logo-pantalla-carga.png` → ancho ~420 px, fondo transparente.

- `aizen-logo-sm.png` → ancho ~120 px, fondo transparente.
- `og-image.png` → **1200 × 630 px** (proporción 1.91:1, estándar Open Graph).

## Próximos pasos

1. Exporta cada imagen con el nombre exacto indicado en las tablas.
2. Sube los archivos a la carpeta correspondiente (puedes arrastrarlos directamente en GitHub o usar `git add`).
3. Los archivos `.gitkeep` en cada carpeta pueden eliminarse una vez que se suban imágenes reales.
4. Verifica que el sitio cargue correctamente abriendo `index.html` en el navegador.
